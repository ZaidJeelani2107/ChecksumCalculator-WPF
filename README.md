# Checksum Calculator (WPF)

A lightweight and user-friendly **WPF desktop application** for generating and comparing file checksums. This tool helps verify file integrity using multiple hashing algorithms through a clean, drag-and-drop interface.

---

## Features

### - Checksum Generator

* Drag and drop a file into the application
* Select from multiple hashing algorithms
* Instantly generate and display the checksum

### - File Comparison

* Drag and drop two files
* Automatically computes checksums for both files
* Displays whether the files are identical or different

### - User Experience

* Simple tab-based interface
* Fast and responsive performance
* Minimal and intuitive design

---

## Technologies Used

* **C#**
* **.NET (WPF)**
* XAML for UI design

---

## Application Overview

### Checksum Tab

* Upload a file via drag-and-drop
* Choose a hashing algorithm
* View the generated checksum

### Compare Tab

* Upload two files
* Generate checksums for both
* Compare and display results clearly

---

## Supported Hash Algorithms

* MD5
* SHA-1
* SHA-256
* SHA-512

---

## How to Use

1. Launch the application
2. Select a tab:

   * **Calculate Checksum Tab** → Generate checksum
   * **Compare Tab** → Compare files
3. Drag and drop file(s)
4. Select a hashing algorithm
5. View results instantly

---
